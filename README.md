This extension generates a branch title based on data from a Zoho Sprints item/task.

------

The pattern of the generated title is:

`{TASK_ID}-{PROJECT_TAG}-{TASK_TITLE}`

Ex.: RF-I1505-PROJECT-Refactor-home-page
